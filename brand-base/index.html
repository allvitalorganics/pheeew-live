The Brand Test · 5 questions

Need a decision? Run it through these.

For any move — a new SKU, a new retailer, a new hire, a new copy line, a freelancer's draft. Answer all five. Any "I don't know" or "kind of" → kill it and reroll.

QUESTION 01
Does it deliver relief?
Not fun, not clinical, not aesthetic-only.
QUESTION 02
Does it sound like a 32-year-old Korean designer in Seoul talks?
The Dawon test.
QUESTION 03
Does it live in the Oasis · Desert · Road · Water lexicon?
Or does it contradict the metaphor.
QUESTION 04
Is it Catalyst-mode or Helper-mode?
If neither, what is it — and is that allowed.
QUESTION 05
Does it inherit our cool-girl unbotheredness — or does it perform?
Performance breaks the brand. Confidence builds it.
Reference · the rules below the kit

If anything in this kit drifts,
these are the non-negotiables.

Compressed. Memorize. Everything in the kit above inherits from these — if a copy line, asset, or decision contradicts what's below, the rule wins.

Position

Pheeew is the relief brand for the people whose bodies lock up.

  • Wedge: Not clinical (Bloom). Not fun (Olipop). Relief — the lane in between.
  • Format: 30mL daily smoothie spout pouch. Not a soda. Not a powder.
  • Function on pack: GUT + BLOAT. Always. Never flavor-only.
SKU architecture

Brand · Function · Flavor.

  • Pheeew · Gut + Bloat · Sunrise Berries (Wild Berries blend)
  • Pheeew · Gut + Bloat · Mojave Mango (Tropical Punch blend)
  • Pheeew · Gut + Bloat · Cold Water Pear (Ginger Pear blend)
  • Future lines extend FUNCTION: Pheeew · Period · _ / Pheeew · Travel · _ / Pheeew · Pro · _
Voice

Cool girl, unbothered.

  • 6–14 word sentences. No exclamation marks.
  • Lowercase or sentence case. SCREAMING UPPER for one-line headlines only.
  • Banned: synbiotic · clinical-grade · doctor-formulated · holistic · wellness journey · besties · girlies · self-care · "real talk" · "spill the tea"
  • Approved: tight · locked up · go · finally · pheeew · relief · oasis · desert · period week · day three
Visual

Studio Mojave.

  • Cream is universal. Cobalt + brick are the loud voices.
  • Playfair italic 900 = display. Caveat Brush = brand statement. Inter = body. Special Elite = function/dose.
  • Banned: Bebas Neue (hypebeast). DM Serif Display (clinical-DTC).
  • Postcard tilts (2–4°) on cards. Single-subject + sky photography. Curved adobe, not glass-box.
Anti-positioning

Named names matter.

  • Not AG1 · we do one thing, not "everything in one"
  • Not Bloom / Lemme · no "synbiotic" on the front
  • Not Olipop / Poppi · we use Brand · Function · Flavor (not Brand · Flavor)
  • Not Goop / Moon Juice · we don't sell rituals or aesthetics — we sell going
  • In spirit: Aesop, Patagonia, Liquid Death, Stüssy, Apartamento
Kill list · top 8

Answer is no without discussion.

  • Skits with recurring character casts (= Performer)
  • Flavor-only SKU naming (= Olipop pricing)
  • "Synbiotic" / "clinical" on the front of pack
  • Pivot to "fun-soda gut health"
  • Drop-hype theatrics — countdown timers, "only 24 hours!"
  • Stylists who've worked on Bloom / AG1 / Lemme / Goop / Daily Harvest
  • "All Vital Organics" anywhere customer-facing
  • Wellness-influencer seeding (anyone with "wellness"/"gut" in handle)
